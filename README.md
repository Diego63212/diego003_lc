</p>

## Utilities

## Config

### Mods should be configured:
<p>BetterStamina: Change most config back or close to 1.0 (vanilla).</p>
<p>FacilityMeltdown: Disable music.</p>
<p>LethalEscape: Allow just a few monster to escape, disable outside spawn.</p>
